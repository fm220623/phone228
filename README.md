# Финансовый менеджер

Приложение для учета денег. Записывай доходы и расходы, следи за лимитами.

## Что умеет
- Записывать доходы и расходы
- Смотреть историю операций
- Ставить лимиты на категории (бюджеты)
- Прикреплять фото чеков
- Включать/выключать уведомления

## Как скачать и установить
1. Перейди в раздел **"Releases"** справа (или сверху на сайте GitHub)
2. Найди файл `bugahaltor.apk` в последнем релизе
3. Скачай его на телефон
4. Установи (может потребоваться разрешить "Установку из неизвестных источников" в настройках)

## Как запустить код
1. Скачай проект
2. Открой в Android Studio
3. Нажми зеленую кнопку "Run"

## Техническое
- Язык: Kotlin
- Хранение данных: GSON (сохранение в JSON-файлы)
- Для списка: RecyclerView
- Фото: загрузка из галереи
- Уведомления: NotificationManager
